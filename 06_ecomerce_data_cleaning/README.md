# E-commerce Sales Data Cleaning

## Overview
This project focuses on data wrangling and cleaning of a messy e-commerce dataset (`messy_ecommerce_sales_data.csv`). The notebook demonstrates various data preprocessing techniques in Pandas, including handling incorrect data types, fixing whitespaces in column names, resolving duplicate IDs, and imputing missing values using category medians.

## Files Included
- `messy_ecommerce_sales_data.csv`
- `notebook.ipynb`
